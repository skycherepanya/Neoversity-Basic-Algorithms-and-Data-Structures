## Завдання 2: Метод Монте-Карло

У завданні реалізовано обчислення визначеного інтеграла функції $f(x) = x^2$ на відрізку $[0, 2]$. 
Було використано три етапи:
1. Побудова графіка для візуалізації площі.
2. Обчислення точного значення за допомогою функції `quad` (аналітичний метод).
3. Обчислення наближеного значення методом Монте-Карло.

### Результати
- **Точне значення (`quad`):** 2.666666666666667 ($8/3$).
- **Метод Монте-Карло:** Отримане значення дуже близьке до точного (залежить від випадковості, наприклад 2.66... або 2.67...).

### Висновок
Метод Монте-Карло показав високу точність при великій кількості ітерацій ($N=100000$). Хоча цей метод є наближеним і дає невелику похибку, він є потужним інструментом для обчислення площ складних фігур та інтегралів, які важко обчислити аналітично.
