# Аналіз графа метро Будапешта

## Завдання 1
Створено граф центральної частини метро (лінії M1, M2, M3, M4).
Вершини — станції, ребра — перегони, вага — час у хвилинах.

## Завдання 2 (DFS vs BFS)
Маршрут: Vörösmarty tér -> Keleti pályaudvar.

1. **BFS (Пошук у ширину)**:
   - Знайшов найкоротший шлях за кількістю станцій (через пересадку на Deák Ferenc tér).
   - Це оптимальний шлях для метро.

2. **DFS (Пошук у глибину)**:
   - Знайшов шлях, але він може бути неоптимальним, оскільки алгоритм йде по гілці до кінця перед тим, як перевіряти інші варіанти.

## Завдання 3 (Дейкстра)
Алгоритм знайшов найшвидший маршрут, враховуючи час проїзду перегонами.
Результат збігся з BFS, оскільки в даній моделі менша кількість станцій також означає менший час.
