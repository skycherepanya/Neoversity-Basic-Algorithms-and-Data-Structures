# Висновки до Завдання 3

Порівняння алгоритмів пошуку підрядка (Боєра-Мура, Кнута-Морріса-Пратта, Рабіна-Карпа) на двох текстових файлах.

## Результати

1. Алгоритм Боєра-Мура:
Найшвидший алгоритм у всіх тестах. Він ефективно пропускає частини тексту, що значно прискорює пошук, особливо для довгих підрядків.

2. Алгоритм Кнута-Морріса-Пратта (KMP):
Показав середній результат. Працює швидше за Рабіна-Карпа, але поступається Боєру-Муру.

3. Алгоритм Рабіна-Карпа:
Найповільніший алгоритм у цьому порівнянні через затрати на обчислення хешу на кожному кроці.

## Підсумок
Найефективнішим для даних текстів є алгоритм Боєра-Мура.
