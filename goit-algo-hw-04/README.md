# Висновки

Я порівняв три алгоритми сортування: Insertion Sort, Merge Sort та Timsort.

## Результати
1. **Insertion Sort**: Працює швидко на маленьких списках, але дуже повільно на великих.
2. **Merge Sort**: Працює стабільно швидко на будь-яких списках.
3. **Timsort**: Найшвидший алгоритм.

## Чому Timsort кращий?
Timsort — це комбінація двох методів (Insert та Merge). Він дуже розумний і використовує вже відсортовані шматочки даних, а також написаний мовою C, що робить його значно швидшим за звичайний код на Python.

**Висновок:** Краще використовувати стандартну функцію `sorted()`, бо вона найефективніша.
